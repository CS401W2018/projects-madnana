Internet

The Internet is a global network that connects computers and devices all around the world.

Web

The web is part of the Internet, made up of websites and pages that users can access with a browser.

How are the Internet and Web different?

The internet is the network of connected computers, and the web works on the internet. The web needs internet to work, but the internet doens't need the web.

ASCII and Unicode

ASCII is a simple system for encoding text using numbers and unicode is a larger system that can handle more complex languages

What is UTF-8

UTF-8 is a popular way to encode text, compatible with ASCII but able to represent many more characters from different languages.

URL

A URL is the address of a specific resource on the web.

Domain Name

A domain name is the unique name of a website.

HTTP

HTTP is the protocol used to transfer data between a web browser and a website.

HTTPS

HTTPS is the secure version of HTTP, which encrypts data to keep it private and safe.

IP Address

An IP address is a unique number assigned to each device on the Internet.

DNS

DNS is a system that translates domain names into IP addresses so you can access websites by typing their names.

W3C Recommendations

W3C recommendations are guidelines that help keep the web working smoothly and consistently across different devices and browsers.

XML, XHTML, HTML, HTML5

XML: A language for organizing and storing data.

XHTML: A stricter version of HTML that follows XML rules.

HTML: The basic language for creating web pages.

HTML5: It is the latest version of HTML, with new features for websites.
