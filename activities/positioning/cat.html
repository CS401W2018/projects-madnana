What is the difference between a block and an inline tag? Give examples of each one.
Block elements have top and bottom margins. Inline elements don't have a top and bottom margin.
Example of block elements: <p>
Example of Inline tag: <span>
What is normal flow in HTML?
It is the way that Block and Inline elements are displayed on a page before any changes are made to their layout.
 What does float do? Why do you need to clear the float?
The float property is used for positioning and formatting content. We need to clear the float property to control the floating elements and prevent from overlapping.
 What do absolute, fixed, and relative do?
Relative - the element is positioned relative to its normal position.
Absolute - the element is positioned absolutely to its first positioned parent.
Fixed - the element is positioned related to the browser window.
 How can you make nice navigation buttons for mobile devices?
We can use a combination of CSS properties like flexbox and media queries.
