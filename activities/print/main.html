Why is the viewport meta tag important?
Because it lets you control the width and scaling of the viewport so that it's sized correctly on all devices
How can you adjust the CSS for printing?
You can use the CSS @media and make a CSS print page, to select how you want your page to be displayed.
When would you want to adjust the CSS for printing?
To adjust layout to take advantage of the size and shape of the paper, and to use different styles to enhance the appearance of your content on paper.
