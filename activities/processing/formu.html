What is JavaScript? How did it get its name?
JavaScript is a programming language used to make web pages interactive. also known as LiveScript, but was renamed JavaScript for marketing purposes to ride on the popularity of Java.

What are the two ways you can run JavaScript?
In the browser, it runs on web pages to make them interactive. On the server, it runs on the server to handle things like databases and API requests (using tools like Node.js).

What was the most popular front-end js library?
jQuery was the most popular, making tasks like DOM manipulation and AJAX simpler.

What is the most popular front-end js library today?
React is the most popular now, used to build dynamic, reusable UI components with high performance.
