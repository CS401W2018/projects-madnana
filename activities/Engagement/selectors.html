CSS Selectors

List out the different ways to select an item.
You can use Tag Name, Descendent, ID and Class.

What is the difference between an ID and a Class?
Id can only be used once, once you validate your html it is going to send an error, however, with class you can reuse it as many times as you want.

What is the difference between a SPAN tag and a DIV tag?
div element is used for block-level organization, it causes linebreaks. The span tag is an in-line element with no semantic meaning, usually meant to style sections of a page

What is the difference between a DIV tag and HEADER, MAIN, FOOTER?
They are all similar to DIV tag and have the same function. Instead of giving div for everything on the page, we now have clear names to separate our contents, and it is not necessary to put ID on every division.
