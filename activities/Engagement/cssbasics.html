Explain the difference between inline, embedded, and external CSS.
Inline styles: used as part of the HTML tag itself.
Embedded styles: Inside of the header of a page, and apply to that entire page.
External style: the sheet is a separate text file that each page in the web site can link to it in order to receive it's instructions.

Explain why you should only use external CSS.
It is easier to keep it uniform and clean, as well as it is easier to update the website and needed to make changes

What is a selector used for?
Its target is the HTML elements on our web pages that we want to style.
